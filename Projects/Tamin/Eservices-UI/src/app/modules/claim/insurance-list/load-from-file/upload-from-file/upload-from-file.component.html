با انتخاب کارگاه، مشخصات کارگاه انتخاب شده در کادرهای مربوطه نمایش داده خواهد شد. سپس با انتخاب فایل های مربوطه و انتخاب گزینه "بارگذاری فایل ها"، فایل های انتخابی به سرویس دهنده مرکزی ارسال شده و در صورت نبود هیچگونه خطا در فایل های ارسال شده، سیستم شما را به مرحله بعدی ("مشاهده اطلاعات لیست") هدایت خواهد کرد. در صورتیکه فایل های ارسال شده دارای خطا باشند، سیستم آنرا شناسایی نموده و پیامی نشان داده خواهد.
